# LangChain

---

## 📁 Files

* `first_agent_in_langchain.ipynb`
  A Jupyter Notebook demonstrating how to create and run a basic LangChain agent.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

* Python 3.8+
* Jupyter Notebook or Jupyter Lab

### Installation

Install the required dependencies:

```bash
pip install langchain openai
```

> You may also need to install additional packages depending on your setup.

### Running the Notebook

1. Clone the repository:

   ```bash
   git clone https://github.com/mehedi-nahi/LangChain.git
   cd LangChain
   ```

2. Start Jupyter:

   ```bash
   jupyter notebook
   ```

3. Open `first_agent_in_langchain.ipynb` and run the cells step by step.

## 🧠 What You'll Learn

* Basics of LangChain
* How to create an agent
* How to connect an LLM
* Running simple tasks using an agent

## 📌 Notes

* Make sure to set your API keys (e.g., OpenAI) before running the notebook.
* This is a beginner-friendly example and can be extended for more advanced use cases.
